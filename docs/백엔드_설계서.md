# 견적 원가 분석 시스템 - 백엔드 설계서

> **프로젝트**: 현대모비스 견적 원가 분석 시스템  
> **버전**: v2.0  
> **작성일**: 2026-04-08  
> **기술 스택**: Spring Boot 3.2 + PostgreSQL 16 + Redis 7 + Nginx 1.25

### 1.2 기술 스택 상세

| 구분 | 기술 | 버전 | 용도 |
|------|------|------|------|
| WAS | Spring Boot | 3.2.4 | REST API 서버 |
| 언어 | Java | 17 (LTS) | 백엔드 개발 |
| ORM | Spring Data JPA | - | 데이터 접근 |
| DB | PostgreSQL | 16 | 주 데이터 저장소 |
| 캐시 | Redis | 7 | JWT 블랙리스트, 세션 |
| 파일저장소 | MinIO | latest | Excel/PDF 원본 파일 |
| 웹서버 | Nginx | 1.25 | 리버스 프록시, 정적 파일 |
| 인증 | JWT | jjwt 0.12.5 | Access/Refresh Token |
| 문서 | SpringDoc OpenAPI | 2.4.0 | Swagger UI |
| 빌드 | Gradle | - | 의존성 관리, 빌드 |
| 컨테이너 | Docker Compose | 3.9 | 로컬/운영 배포 |
| AI | Claude API | claude-opus-4-6 | 인사이트 채팅 |
| Excel | Apache POI | 5.2.5 | Excel 파싱 |
| PDF | Apache PDFBox | 3.0.2 | PDF 파싱 |

## 5. 인증/보안

### 5.1 JWT 인증 흐름

```
1. 로그인: POST /api/v1/auth/login
   → 사번 + 비밀번호 검증
   → Access Token (1시간) + Refresh Token (7일) 발급

2. API 요청: Authorization: Bearer {accessToken}
   → JwtAuthenticationFilter에서 검증
   → SecurityContext에 사용자 정보 설정

3. 토큰 갱신: POST /api/v1/auth/refresh
   → Refresh Token 검증 → 새 Access Token 발급

4. 로그아웃: POST /api/v1/auth/logout
   → Redis에 토큰 블랙리스트 등록
```

### 5.2 역할 기반 접근 제어 (RBAC)

| 역할 | 코드 | 권한 |
|------|------|------|
| 관리자 | ADMIN | 전체 기능 + 수식 직접 편집 + 변경 요청 승인/반려 |
| 일반 사용자 | USER | 파싱/검증/분석/비교 + 수정 요청 제출만 |

### 5.3 계정 잠금 정책

- 5회 연속 로그인 실패 시 계정 자동 잠금
- Redis에 실패 횟수 저장 (TTL: 30분)
- 관리자가 수동 해제 가능
